## The North Star

**Design system goal:** *Reduce cognitive load per decision.*

Not "pretty screens". ERP users judge you by:

| Metric              | What It Measures                           |
| ------------------- | ------------------------------------------ |
| **Time-to-find**    | How fast can I locate what I need?         |
| **Time-to-confirm** | How fast can I verify the data is correct? |
| **Time-to-post**    | How fast can I complete the transaction?   |
| **Error rate**      | How often do I make mistakes?              |
| **Reversibility**   | Can I undo if something goes wrong?        |

**AXIS Mantra:**

> **"Clarity at speed. Confidence by proof."**

### Law C â€” Interaction Beats Decoration

If a user can finish a task **without touching the mouse**, that's premium.

| Principle          | Implementation                                         |
| ------------------ | ------------------------------------------------------ |
| **Keyboard-first** | Every action has a keyboard shortcut                   |
| **Tab-through**    | Forms can be completed via Tab + Enter                 |
| **Quick search**   | `/` activates table search, `âŒ˜K` opens command palette |
| **Predictive**     | Smart defaults reduce keystrokes                       |

**The QuickBooks Test (Quorum):** Can a bookkeeper complete a task in under 3 clicks?

**The Bloomberg Test (Cobalt):** Can a power user complete a task without leaving the keyboard?

## Part X: Governance â€” Preventing Design Drift

### Non-Negotiable Rules

| Rule                          | Enforcement                | Violation              |
| ----------------------------- | -------------------------- | ---------------------- |
| **No ad-hoc colors**          | Lint: ban raw hex/rgb      | `bg-[#ff0000]`         |
| **No local component copies** | Lint: ban `../ui/` imports | Copying Button locally |
| **All variants via CVA**      | Code review                | Inline variant logic   |
| **All sizes via tokens**      | Lint: ban arbitrary values | `text-[15px]`          |
| **All states share spec**     | Design review              | Custom state styling   |

### Quality Gates (Automated)

```typescript
// eslint-plugin-axis/no-raw-colors.ts
export const noRawColors = {
  meta: { type: "problem" },
  create(context) {
    return {
      Literal(node) {
        if (typeof node.value === "string") {
          // Ban hex colors
          if (/^#[0-9a-f]{3,8}$/i.test(node.value)) {
            context.report({ node, message: "Use semantic tokens, not hex colors" });
          }
          // Ban rgb/rgba
          if (/^rgba?\(/.test(node.value)) {
            context.report({ node, message: "Use semantic tokens, not rgb colors" });
          }
        }
      },
    };
  },
};

// eslint-plugin-axis/workspace-imports.ts
export const workspaceImports = {
  meta: { type: "problem" },
  create(context) {
    return {
      ImportDeclaration(node) {
        const source = node.source.value;
        // Ban local UI imports
        if (source.includes("../ui/") || source.includes("./ui/")) {
          context.report({
            node,
            message: "Import from @workspace/design-system, not local copies",
          });
        }
      },
    };
  },
};
```
